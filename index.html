Estadística Básica: Medidas de Tendencia Central y Dispersión

1. Media

La media es el promedio de un conjunto de datos.

Fórmula: Media = Σxi / n

Ejemplo: Para las edades 20, 25, 30, 35, y 40: Media = (20 + 25 + 30 + 35 + 40) / 5 = 30

Ejercicio:

Encuentra la media de los números: 8, 12, 15, 10, 17.

2. Mediana

La mediana es el valor central de un conjunto de datos ordenado.

Fórmula: Valor central si n es impar, promedio de los dos valores centrales si n es par.

Ejemplo: Para los datos 15, 18, 22, 26, y 30, la mediana es 22.

Ejercicio:

Encuentra la mediana de los valores: 10, 14, 18, 20, 22, 25, 30.

3. Moda

La moda es el valor que más se repite en un conjunto de datos.

Ejemplo: Para los datos 4, 5, 5, 6, 7, 8, 8, 9, la moda es 5 y 8.

Ejercicio:

Encuentra la moda de los valores: 6, 9, 6, 11, 12, 9, 15.

4. Rango

El rango es la diferencia entre el valor máximo y el mínimo.

Fórmula: Rango = Valor máximo - Valor mínimo

Ejemplo: Para los datos 5, 10, 15, 20, y 25: Rango = 25 - 5 = 20

Ejercicio:

Encuentra el rango de los valores: 8, 12, 20, 4, 15.

5. Desviación Media

La desviación media mide cuánto se desvían, en promedio, los valores de un conjunto de datos respecto a la media.

Fórmula: Desviación Media = Σ|xi - x̄| / n

Ejemplo: Para los datos 10, 15, 20, 25, y 30, la desviación media es 6.

Ejercicio:

Calcula la desviación media para los datos: 4, 8, 12, 16, 20.

6. Desviación Estándar

La desviación estándar indica cuánto varían los datos respecto a la media.

Fórmula: σ = √(Σ(xi - x̄)² / n)

Ejemplo: Para los datos 5, 10, 10, 15, y 20, la desviación estándar es aproximadamente 5.1.

Ejercicio:

Calcula la desviación estándar para los datos: 3, 7, 7, 10, 13, 17.
